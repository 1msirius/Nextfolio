<ProjectOverview>
These animations start from a single text‑to‑image prompt that generates a base frame. Each subsequent frame runs through an image‑to‑image diffusion pipeline — reconditioning on the same prompt while gradually injecting noise. The result is a controlled, chaotic melting: a visual degradation that blurs the boundary between creation and entropy.
</ProjectOverview>
